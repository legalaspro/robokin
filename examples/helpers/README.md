# Example Helpers

Shared utilities used by the example scripts.

## Files

| File | Description |
|------|-------------|
| `so101_lerobot.py` | `SO101LeRobotArm` wrapper around the LeRobot `SO100Follower` driver. Provides a simple interface for reading/writing joint positions, PID configuration, and background servo threads. Used by the `lerobot_real_arm.py` examples. |
